Top 3 Ideas

1.

For computer users
Who sit at a desk for hours at a time,
LookAway
Is a web page that will play a designated audio clip, and display a prompt to follow the 20-20-20 rule. Every 20 minutes, look at an object roughly 20 minutes away, for 20 seconds.
Unlike relying on setting a manual reminder through an alarm program on your phone,
Our product is web based, and can be used on any device that has a web browser.

2.

For personal consumers
Who are getting ready to shop or pay bills, but need to verify they have enough funds,
BudgetCheck
Is a web page that you can enter your total budget amount, and then enter each individual amount you want to automatically subtract from that budget to make sure you will have enough funds.
Unlike Microsoft Excel,
Our product doesn't require the user to create complicated formulas or color formatting, as it is preprogrammed, and available on any device that has a web browser.

3.

For parents
Who have multiple children and need to set a schedule for children to take turns on a video game console or any device,
WhoseTurnIsIt
Is a web based application that will allow the user to enter each child's name, and the amount of time they are allowed on a system. The application will then create a queue, and alarm the parent when the next child's turn has arrived Unlike using a home assistant such as google home,
Our product will allow you to set the number of turns each child gets, and how long each individual child gets.
